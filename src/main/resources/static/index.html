Projet CI/CD Spring Boot

Cette application a été construite avec Spring Boot, empaquetée avec Maven, conteneurisée avec Docker et déployée automatiquement avec GitHub Actions.

Le pipeline exécute les tests, construit l’image Docker, publie l’image puis déploie l’application sur un serveur Ubuntu à l’aide d’un runner auto-hébergé.
